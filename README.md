# VS_code_customisation

a simple collection of codes to customise your vs code looks.

## How to use it ?
1. open a md file here
2. copy the json code
3. paste it in your workspace by adding a folder `.vscode`
    > paste in `.vscode/settings.json`

###  dont add outer braces if you already have somethings written in your `.vscode/settings.json` file.
